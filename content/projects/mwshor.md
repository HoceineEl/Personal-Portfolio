## Mwshor – SaaS Project Management Platform

### Overview

Mwshor is a multi-tenant project management platform offering Kanban boards, timelines, and task management. Scalable for any industry with client tracking through personalized panels.

### Key Features

- **Kanban Boards** – Visual task management
- **Timeline Views** – Project scheduling and deadlines
- **Task Management** – Assign, track, and complete tasks
- **Client Panels** – Personalized dashboards for clients
- **Multi-Tenant** – Separate workspaces per organization
- **Team Collaboration** – Real-time updates and comments

### Technologies Used

- **Laravel** – Backend framework
- **Livewire** – Dynamic interfaces
- **Alpine.js** – Frontend reactivity
- **FilamentPHP** – Admin dashboard
- **Tailwind CSS** – UI styling
