## WiserPocket – Financial Management Platform

### Overview

WiserPocket is a comprehensive financial management platform that helps users track expenses, set budgets, and achieve financial goals with AI-powered insights.

### Key Features

- **AI Chat Assistant** – Get financial advice through conversational AI
- **Expense Tracking** – Categorize and monitor all transactions
- **Budget Management** – Set and track spending limits
- **Analytics Dashboard** – Visual charts and insights
- **Voice Input** – Add transactions using voice commands
- **Goal Tracking** – Set and monitor financial goals

### Technologies Used

- **Laravel** – Backend framework
- **Livewire** – Real-time updates
- **FilamentPHP** – Admin dashboard
- **Tailwind CSS** – UI styling
- **Paddle** – Subscription billing
