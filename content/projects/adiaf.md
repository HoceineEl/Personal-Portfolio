## Adiaf – Hajj Management SaaS System

### Overview

Adiaf is a comprehensive Hajj management system with multi-tenant architecture and role-based access. Manages the entire pilgrimage workflow from registration to completion.

### Key Features

- **Multi-Tenant Architecture** – Separate spaces per organization
- **Role-Based Panels:**
  - Company Dashboard
  - Center Management
  - Admin Controls
  - Attendant Interface
  - Guide Portal
  - Pilgrim App
- **Pilgrim Registration** – Complete registration workflow
- **Group Management** – Organize pilgrims into groups
- **Schedule Management** – Track all Hajj activities

### Technologies Used

- **Laravel** – Backend framework
- **Livewire** – Real-time features
- **FilamentPHP** – Multi-panel admin
- **Tailwind CSS** – Responsive design

### Client

Built for Ashraqat Company.
